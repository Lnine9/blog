# Dollars 实时聊天软件
### 项目简介
- **项目描述** JavaEE课程的课程设计，要求是实现一个实时聊天软件。
- **项目时间** 2022年6月5号-2022年6月9号

### 主要技术
后端： springboot + netty + mybatis-plus + mysql
前端： react + antd + styled-components + axios + websocket
