# 采购数据挖掘分析整合系统项目
- **需求单位** 重庆港澳大家有限公司
- **参与身份** 项目负责人，核心开发人员
- **项目时间** 2022年4月-至今
- **项目描述** 通过对中国政府网、公共资源网的招投标网站进行数据抓取与与分析，挖掘招投标关键信息，补充大家软件公司招投标公告资源库。项目以稳定运行一年，累积抓取数据上百万条。
- **主要技术** python + scrapy + django + mysql
- **主要负责** 需求分析、系统架构开发、编写公共方法库、爬虫开发、服务器运维。
- **项目经验** 熟练掌握了python以及网络爬虫的开发，使用django开发服务端程序。对于网络请求，服务器渲染等有了更清晰的理解。熟悉linux环境下的应用部署。
